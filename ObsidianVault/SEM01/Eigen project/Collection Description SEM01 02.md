Inleiding

Deze collectie hoort bij semester 1 van de HBO-opleiding Fontys ICT. Het individuele project dat ik hiervoor uitvoer is Bewegingsanalyse Powerliften. Het project richt zich op het ontwikkelen van een prototype meetinstrument dat coaches en/of fysiotherapeuten inzicht geeft in de bewegingen die iemand maakt tijdens het powerliften.
